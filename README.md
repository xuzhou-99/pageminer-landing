# PageMiner 官方网站

> 🚀 智能网页数据提取助手的官方展示网站

## 📖 项目简介

PageMiner是一款强大的浏览器扩展，帮助用户通过可视化操作快速、准确地从任意网页中提取结构化数据。本网站作为产品的官方展示页面，采用现代化的设计风格，提供完整的产品介绍、功能演示和用户引导。

## ✨ 主要特性

- 🎨 **现代化设计** - 参考simplescraper.io的设计风格，简洁美观
- 📱 **响应式布局** - 完美适配桌面端、平板端和移动端
- 🚀 **高性能** - 优化的CSS和JavaScript，流畅的用户体验
- 🎯 **用户友好** - 清晰的信息架构和直观的操作流程
- 🔧 **易于维护** - 模块化的代码结构，便于后续优化
- 🎥 **功能演示** - 基于实际插件代码的交互式演示
- 🔍 **技术展示** - 真实的功能特性和代码示例

## 🚀 快速开始

### 1. 克隆项目
```bash
git clone https://github.com/your-username/pageminer-landing.git
cd pageminer-landing
```

### 2. 本地预览
由于是纯静态网站，您可以直接在浏览器中打开文件：

- 双击 `index.html` 查看主页面
- 双击 `demo.html` 查看演示页面

### 3. 功能演示
- **主页面**：产品介绍、功能特性、价格方案等
- **演示页面**：基于实际插件代码的功能演示和交互体验
- **观看演示**：主页面中的"观看演示视频"按钮直接链接到demo.html

### 4. 使用本地服务器（推荐）
```bash
# 使用Python 3
python -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后在浏览器中访问 `http://localhost:8000`
